# Goal 任务计划 GoalPlan

本文件保存业务任务、证据、缺陷和阻塞明细。阶段状态的唯一数据源是 `workflow/state.json`；下面的状态区由 `python3 scripts/workflow.py sync` 生成，`index.html` 也读取同一份生成数据。

阶段推进只检查“当前阶段”的门禁。未来阶段的空白或待开始项不参与当前阶段判断；当前阶段可以是 `pending`、`in_progress`、`blocked`、`completed` 或 `not_applicable`。阻塞项必须记录恢复动作，解除后才能恢复推进。
